# 📄 Folio – PDF Editing Suite

A browser-based PDF editor built with **HTML, CSS, and JavaScript**. It allows users to edit, merge, split, compress, convert, sign, and organize PDF files directly in the browser without uploading them to any server.

## ✨ Features

* Merge and split PDFs
* Organize, rotate, and crop pages
* Compress PDF files
* Convert PDF to HTML, Text, CSV, and Images to PDF
* Add text, watermarks, page numbers, and signatures
* Compare and protect PDF files
* AI-powered PDF summarization and translation (Anthropic API)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript 
* pdf-lib
* PDF.js
  

## 🚀 How to Run

1. Download or clone the project.
2. Keep all project files in the same folder.
3. Open "index.html" in any modern browser, or use the VS Code **Live Server** extension.

## 🔒 Privacy

* 100% browser-based processing
* No file uploads
* No data storage
* Works offline (except AI features)
